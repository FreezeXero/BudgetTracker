# Budget Tracker

A personal budget tracking console application built with Java.

## Features
- Track income and expenses
- Categorize expenses (Food, Rent, Transport, Entertainment, Other)
- View all transactions
- View spending breakdown by category
- Calculate total income, expenses, and balance

## Technologies Used
- Java
- ArrayList data structures
- Scanner for user input

## How to Run
1. Clone the repository
2. Open in IntelliJ IDEA or any Java IDE
3. Run Main.java
4. Follow the menu prompts to manage your budget

## Project Structure
- Uses parallel ArrayLists to store transaction data
- Menu-driven interface with 5 options
- Category-based expense tracking and analysis
